# Luma Core Demo — Changelog

## v1.0.0 — Initial Release (2025-12-03)

### Summary
This is the first public release of the **Luma Core Demo**, the reference implementation of the XIP Protocol.  
It provides a working example of XIP behaviour across multiple hubs using an offline-first PWA.

---

### Key Features

#### 1. Reference Receipt Engine
- Implements XIP receipt envelopes  
- Hash-chained event ledger  
- Deterministic offline behaviour  
- Pluggable sync adapter  

#### 2. Multi-Hub Demonstration
- Finance hub  
- Government hub  
- Cannabis/Health hub (simplified optional flow)  
- Unified event generation across hubs  

#### 3. Offline-First PWA
- Installable application  
- Runs with no network  
- Local chain storage  
- Background sync queue  

#### 4. Developer-Friendly Structure
- Modular source code  
- Clearly separated domain logic  
- Debug window for receipts  
- Easy to extend or reskin
