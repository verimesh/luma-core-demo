# Contributing to Luma Core Demo

Thank you for your interest in the Luma Core project.

At this stage, this repository is provided for demonstration and evaluation only.
Unsolicited pull requests or code contributions may be closed without review.

## 1. Public Contributions

Public issue reports and suggestions are welcome.

However:

- you should not submit code, patches, or pull requests unless requested,
- any submitted content becomes the property of the Luma Core project,
- you retain no authorship, ownership, or royalty claims.

## 2. Formal Collaboration

Any deeper technical work requires:

- a signed Non-Disclosure Agreement (NDA),
- a signed Contributor or Collaboration Agreement,
- explicit approval from the project maintainer.

## 3. IP and Licensing

By submitting any form of feedback—issues, ideas, or comments—you agree that:

- the Luma Core project may use or ignore your input,
- no compensation or acknowledgement is required,
- no licence or ownership claim is transferred to you.
