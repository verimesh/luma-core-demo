## Scope

This repository provides a demonstration-only implementation of selected Luma Core / XIP Protocol
concepts. It is not a production system and should not be deployed as such.

There are:

- no backend services,
- no live user accounts,
- no production keys,
- no sensitive data stored in this repository.

## Reporting Issues

If you identify a potential security issue in this demo:

1. Do **not** publicly post exploit details.
2. Contact the Luma Core project via the contact page at:  
   https://verimesh.net

Please describe:

- the nature of the issue,
- what part of the demo is affected,
- any steps necessary to reproduce it.

## Intended Use

This demo is provided for evaluation purposes only. Any modification, redeployment, or integration
into other systems is your responsibility and may be subject to additional licensing requirements.

## No Bug Bounty

There is currently no public bug bounty programme for this demo. Reports are still welcome and help
improve the broader Luma Core ecosystem.
